# AI Support Ticketing Backend

Backend API for an AI-powered support ticket system that analyzes customer issues,
classifies them, and assists support teams with AI-generated responses.

This project is being built incrementally with production-grade practices.

---

## 🚀 Tech Stack

- Python 3.12+
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- OpenAI API
- Alembic (Migrations)
- Pytest



## 📌 Notes

- Secrets are managed using environment variables (`.env`)
- `.env` files are intentionally excluded from version control
- API design follows a service-based architecture
